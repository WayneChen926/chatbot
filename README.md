# chatbot
### Line 天氣預報機器人
- 取得氣象資料
- 整理後判斷關鍵字地區，透過Line回覆

### 使用教學
- 在內容打上地區會回覆您，該地區的天氣預報。

![](https://i.imgur.com/yXD4lbt.png)

- 還可以使用location，找到你所在的城市區域，進行天氣預報。

![](https://i.imgur.com/UXhUDzs.jpg)

### QR code
- 有興趣可以掃以下QR code 使用看看

![](https://i.imgur.com/pMhahKz.png)
